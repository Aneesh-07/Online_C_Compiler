Welcome to the Online C Compiler

Hello there! I'm Aneesh Tripathi, and this is my personal project: the Online C Compiler.

If you're curious about coding in C or you're looking to quickly compile and test your C programs, you're in the right place.

As part of my journey to learn Flask, HTML, and CSS, I've developed this web-based compiler to make the process of writing and running C code more convenient.

Feel free to explore the compiler page where you can write your C code, submit it for compilation, and see the results in real time.

Thank you for checking out my project, and I hope you find it helpful and informative!
